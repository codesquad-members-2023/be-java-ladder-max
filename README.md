Naming convention
- 클래스, 메소드, 변수 의 이름만 봐도 이게 무슨 일을 할 지 짐작할 수 있게끔 명명한다.
- Java는 Camel Case를 따르도록 한다.
  - Camel Case : 첫번째 단어는 소문자로 시작, 이후 단어들의 첫 글자는 대문자로 쓰는 방식이다.
    - 클래스와 변수의 이름은 명사, 메소드의 이름은 동사 로 시작하는 경우가 많다.
