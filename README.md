## ✔️ 기능 요구사항

### 1단계
- [X] n명의 사람을 입력하는 기능
- [X] n개의 사다리 개수를 입력하는 기능
- [X] 랜덤으로 사다리를 생성하는 기능
    - 사다리가 있으면 `-` 표시, 없으면 `" "`(공백문자) 표시, 양옆에는 `|`로 표시
- [X] 사다리를 출력하는 기능

### 2단계
#### 입력 기능
- [ ] 플레이어의 이름을 입력하는 기능(최대 5자)
  - [ ] 예외처리: 
    - 이름이 6자 이상일 시
    - 쉼표로 구분하지 않았을 시
    - 1명 참여 시(2명 이상 참여 가능)

#### 출력 기능
- [ ] 사다리 출력 시 이름도 같이 출력하는 기능

#### 사다리 기능
- [ ] 사다리 생성 시 라인이 겹치지 않도록 생성하는 기능
- [ ] 예외 처리:
  - 사다리 높이가 1이하일 시
