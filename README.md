# Java Ladder

- 1 단계 : Ladder 객체를 위한 클래스 생성 및 구현
  - 사다리의 라인만 표시하는 2차원 배열 구현 
  - n명의 사람과 m개의 사다리 개수 = (n - 1) * m 짜리 배열로 구현 후 출력 시 | 붙여줌.
  - 랜덤값으로 라인이 있는지 없는지만 표시
