## 콘솔 기반 사다리 게임 실행 결과

```shell
참여할 사람 이름을 입력하세요. (이름은 쉼표(,)로 구분하세요)
pobi,hounx,crong,jk
실행 결과를 입력하세요. (결과는 쉼표(,)로 구분하세요)
1000,2000,3000,4000
최대 사다리 높이는 몇 개인가요?
5
pobi  hounx crong jk

|     |-----|     |
|-----|     |     |
|     |-----|     |
|     |-----|     |
1000 2000 3000 4000 
결과를 보고 싶은 사람은?
all : 전체 결과 보기, 춘식이 : 프로그램 종료, 이름 : pobi  hounx crong jk   
all

jk    : 4000
```
